#SpringBoot快速搭建模板

**目前仅使用于mysql**

## 所用技术
1. SpringBoot 1.5.10
2. Thymeleaf 3
3. Hikari datasource
4. Mybatis
5. Mybatis分页插件
6. Mybatis通用Mapper
7. Mybatis generator
8. 统一异常处理/统一Json对象
